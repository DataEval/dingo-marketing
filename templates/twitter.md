# Twitter/X Post Template

## Style Rules

- Generate three formats: (1) English thread, (2) single English tweet, (3) single Chinese tweet
- Thread: 3-5 tweets, each under 280 characters
- Single tweet: under 280 characters, most impactful summary
- Include links: SaaS URL (if exists), install command, repo URL
- Minimal emoji usage: only link/package indicators allowed
- Hashtags from project config, appended at end of last tweet or single tweet
- Professional tone, no hype

## Thread Structure

```
**1/N**
{Project} {version} released — {headline}.

{One sentence on what it does.}

{saas_url}
{install_command}

**2/N**
[SaaS highlights or key new features — bullet list]

**3/N**
[Technical highlights — architecture, metrics, integrations]

**N/N**
{install_command}
{repo_url}
{hashtags}
```

## Single Tweet Structure

```
{Project} {version} — {headline}. {2-3 key features}. {saas_url or repo_url} {install_command}
```

## Chinese Tweet

- Write entirely in Chinese
- Same content density as single English tweet
- Include links at end
